## include

* `ademco_event.h` 

    定义了安定宝事件码及所需的结构体

* `ademco_detail.h` 

    定义了各种辅助函数

* `ademco_packet.h` 

    定义了可用于安定宝协议封包、解包的结构体与相应函数

## examples

* `Windows` 下使用 `VS 2019 Community` 打开 `examples.sln` 进行编译。
* `Linux/MacOS` 下执行 `build.sh` 编译示例程序，结果放在 `build` 文件夹内。
* 如需编译 `javademo` 请阅读 [javademo-README.md](examples/javademo-README.md)
